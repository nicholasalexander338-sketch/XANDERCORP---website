## 🚀 Deployment (GitHub Pages)
1. Go to **Settings → Pages**  
2. Under **Build and deployment**, choose:  
   - **Source:** Deploy from branch  
   - **Branch:** `main`  
   - **Folder:** `/ (root)`  
3. Save and wait 1–2 minutes for the live link to activate.
